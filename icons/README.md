# 图标文件说明

这个文件夹需要包含以下图标文件：

- `icon16.png` - 16x16 像素图标
- `icon32.png` - 32x32 像素图标  
- `icon48.png` - 48x48 像素图标
- `icon128.png` - 128x128 像素图标

## 图标要求

- 格式：PNG
- 背景：透明或白色
- 设计：简洁明了，体现信息提取功能
- 颜色：建议使用蓝色系，与扩展主题一致

## 临时解决方案

如果暂时没有图标文件，可以：

1. 使用在线图标生成器创建简单图标
2. 从图标库下载合适的图标
3. 使用占位符图片（纯色方块）

## 推荐图标设计

- 链接图标 + 放大镜
- 数据提取图标
- 信息收集图标
- 网络爬虫图标

图标文件准备好后，Chrome扩展就可以正常显示了！
